# Local-property-tax-strategy-in-Marseille
Academic excersise using prospective methods for economic decision-making in the framework of Master course.

Purpose: The purpose of the project is to ask you to work as an economist advising a real decision-maker
under uncertainty. The central question of the course has been the following: how should economists support
decision-making when the future cannot be reduced to a single forecast? This project asks you to answer that
question in practice.

Context: The City of Marseille must decide whether, over the next three years, it should rely primarily on general rate
increases, on property-tax base modernization and vacancy mobilisation, or on a mixed strategy to strengthen
local property-tax revenue

Data utilized: 
- REI: local direct-tax elements
  <https://www.data.gouv.fr/datasets/impots-locaux-fichier-de-recensement-des-elements-dimposition-a-la-fiscalite-directe-locale-rei-4>
- LOVAC: vacant private dwellings
  <https://www.data.gouv.fr/datasets/logements-vacants-du-parc-prive-par-commune-departement-region-france>
